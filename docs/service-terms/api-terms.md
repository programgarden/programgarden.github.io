# **E. 시스템 트레이딩 및 외부 API 연동 약관**

**요약:** 본 약관은 ProgramGarden 서버 환경에서 외부 거래소나 증권사의 API를 연동하여 시스템 트레이딩을 수행할 때 적용되는 규정입니다. 외부 API 키 관리 책임, 외부 서비스 장애에 대한 면책, 그리고 자동매매 결과에 대한 사용자 책임을 명시합니다.

---

## **1. 외부 API 연동 및 서비스의 성격**

### 1. 서비스의 제공 범위
ProgramGarden은 사용자가 작성한 알고리즘이나 트레이딩 봇을 24시간 구동할 수 있는 **서버 인프라(SaaS)**를 제공합니다. 서버 실행은 Google Cloud 기반의 클라우드 환경 또는 **로컬 실행 환경(Personal Local Server)**을 통해 이루어집니다. 회사는 직접적인 매매 신호를 제공하거나, 자체적인 트레이딩 API를 제공하지 않습니다. 사용자는 본인의 계정으로 외부 거래소(암호화폐 거래소, 증권사 등)의 API를 직접 연동하여 서비스를 이용해야 합니다.

### 2. 연동 증권사
현재 ProgramGarden이 지원하는 외부 증권사 API 연동은 다음과 같습니다:
*   **LS증권:** 계좌 잔고 조회, 주문 내역 조회, 매매 주문 실행

향후 추가 증권사/거래소 연동이 확대될 수 있습니다.

### 3. 외부 서비스와의 관계
ProgramGarden은 사용자가 연동하는 외부 거래소나 증권사와는 독립된 별개의 서비스 제공자입니다. 외부 거래소/증권사의 정책 변경, API 사양 변경, 서비스 점검, 시스템 장애 등으로 인해 ProgramGarden 내에서의 정상적인 구동이 불가능해질 수 있으며, 이에 대해 회사는 책임을 지지 않습니다.

---

## **2. API 키 관리 및 보안**

### 1. API 키의 보관 및 관리
사용자는 외부 거래소/증권사 접속을 위해 필요한 API Key(앱키, 시크릿키 등)를 안전하게 관리해야 할 의무가 있습니다. 회사는 사용자가 서버에 저장한 API Key를 **Google Cloud 암호화 시스템**으로 강력하게 암호화하여 보관합니다. 단, 사용자의 부주의(코드 내 키 하드코딩, 계정 유출 등)로 인해 발생한 키 유출 사고에 대해서는 책임을 지지 않습니다.

### 2. 권한 설정 권고
사용자는 외부 API 키 발급 시, 트레이딩에 필요한 최소한의 권한(주문, 조회 등)만을 허용해야 하며, '출금(Withdrawal)' 권한은 절대 활성화해서는 안 됩니다. 출금 권한이 포함된 API 키를 사용하여 발생한 자산 탈취 사고에 대해 회사는 어떠한 책임도 지지 않습니다.

---

## **3. 외부 서비스 장애 및 면책**

### 1. 외부 API 오류
외부 거래소의 API 서버 불안정, 응답 지연, Rate Limit(요청 제한) 초과, 긴급 점검 등으로 인해 주문이 체결되지 않거나 봇이 중단되는 경우, 이는 회사의 귀책사유가 아니며 회사는 이에 대한 손해배상 책임을 지지 않습니다.

### 2. 네트워크 및 인프라
회사는 안정적인 서버 환경을 제공하기 위해 노력하지만, 인터넷 망 사업자의 장애, 천재지변, 디도스(DDoS) 공격 등 불가항력적인 사유로 인한 서비스 중단에 대해서는 면책됩니다.

---

## **4. 자동매매 및 투자 결과에 대한 책임**

### 1. 투자 결과의 귀속
ProgramGarden 서버 상에서 구동되는 자동매매 시스템, 트레이딩 봇, 또는 기타 알고리즘의 실행 결과로 발생하는 모든 이익과 손실은 전적으로 사용자 본인에게 귀속됩니다. 회사는 사용자가 작성한 코드의 로직이나 투자 판단에 관여하지 않으며, 이에 대한 결과도 책임지지 않습니다.

### 2. 코드 오류 및 버그
사용자가 직접 작성하거나 업로드한 코드의 버그, 로직 오류, 예외 처리 미비로 인해 발생하는 오작동(중복 주문, 주문 누락, 의도치 않은 매매 등)에 대한 책임은 전적으로 사용자에게 있습니다. 사용자는 실제 자산을 운용하기 전에 충분한 백테스트와 모의투자를 통해 시스템의 안정성을 검증할 의무가 있습니다.

---

## **5. 로컬 실행 환경 (Personal Local Server)**

### 1. 서비스 개요
ProgramGarden은 클라우드 실행 외에, 사용자의 로컬 머신에서 워크플로우를 실행할 수 있는 **Personal Local Server**를 제공합니다. 로컬 실행 서버는 ProgramGarden 데스크톱 앱에 내장되어 제공됩니다.

### 2. 보안
로컬 실행 서버는 **사용자의 컴퓨터 내부에서만 접근 가능**하며, 모든 요청은 **Google 인증 토큰**으로 인증됩니다.

### 3. 사용자 책임
로컬 실행 환경의 보안(운영체제 업데이트, 방화벽 설정, 악성코드 방지 등)은 전적으로 사용자의 책임입니다. 로컬 환경에서 발생하는 보안 사고, 데이터 손실, 시스템 장애에 대해 회사는 책임을 지지 않습니다.
